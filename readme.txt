Hola!

Bienvenido a esta versión de Flappy Bird en Pygame!!
Su jugabilidad es exactamente igual a la del juego original, debes ayudar a que el pajaro, 
supere la mayor cantidad de obstaculos posibles.

Aviso!!
Asegurate de tener las librerias "pygame" y "random" para poder disfrutar de este increible juego

Instrucciones:
1. Crear una carpeta, esta carpeta será la que contenga el codigo y todos los archivos necesarios para su ejecución.
2. Descargar el archivo "main.py".
3. Descargar cada una de las imagenes que encontrarás en la carpeta "imagenes".
4. Crea una carpeta llamada "imagenes" y guarda todas las imagenes.
5. Crea una carpeta, en esta deberás guardar, el archivo "main.py" y la carpeta "imagenes".
6. Sube esa carpeta a Visual Studio Code, o a un editor de texto que pueda ejecutar el código.
7. Corre el archivo "main.py"
8. A JUGAR!
